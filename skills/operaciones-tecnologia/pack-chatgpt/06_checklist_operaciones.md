# Checklist - Operaciones

- [ ] Ambiente identificado.
- [ ] Impacto y urgencia definidos.
- [ ] Estado inicial capturado.
- [ ] Prechecks realizados.
- [ ] Backup/rollback considerado.
- [ ] Permisos y secretos protegidos.
- [ ] Comandos revisados y acotados.
- [ ] Logs/evidencia guardados.
- [ ] Prueba de humo definida.
- [ ] Comunicacion y cierre documentados.
