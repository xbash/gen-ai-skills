# Checklist - Revision de notebook Jupyter

## Contexto
- [ ] Curso o modulo identificado.
- [ ] Tema del notebook claro.
- [ ] Enunciado o pauta considerada si existe.
- [ ] Nivel academico asumido o declarado.

## Estructura
- [ ] Celdas Markdown y codigo mapeadas.
- [ ] Marcas `# codigo N` y `# texto N` identificadas.
- [ ] Celdas incompletas sin marca detectadas.
- [ ] Dependencias entre celdas revisadas.
- [ ] Orden de ejecucion coherente.

## Codigo
- [ ] Imports necesarios identificados.
- [ ] Variables definidas antes de usarse.
- [ ] Rutas, archivos o entradas revisadas.
- [ ] Codigo propuesto listo para copiar.
- [ ] Comentarios pedagogicos incluidos.
- [ ] No se agregan librerias injustificadas.

## Texto
- [ ] Respuestas Markdown listas para pegar.
- [ ] Explicaciones conectadas con el curso.
- [ ] No se inventan resultados ni conclusiones.
- [ ] Limitaciones declaradas si falta informacion.

## Reproducibilidad
- [ ] Datos o archivos requeridos identificados.
- [ ] Salidas esperadas claras.
- [ ] Errores potenciales detectados.
- [ ] Supuestos documentados.
- [ ] Recomendaciones finales incluidas.
