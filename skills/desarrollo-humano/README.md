# Dominio: Desarrollo Humano y Sociedad

## Descripcion
Dominio orientado a ciencias sociales, desarrollo humano, cultura, politica, desigualdad, organizaciones, comunidad, datos sociales y analisis critico de fenomenos sociales. Sirve para estudiar problemas sociales, disenar marcos de analisis, revisar politicas publicas, interpretar datos y proponer intervenciones prudentes.

No reemplaza investigacion empirica, trabajo profesional de campo, evaluacion clinica, asesoria legal ni decision publica vinculante. Debe reconocer contexto, evidencia, poder, desigualdad y limites.

## Recomendacion de uso
Para ChatGPT se recomienda usar `pack-chatgpt`, que fusiona el dominio en 6 archivos y deja margen para textos, datos, pautas o antecedentes del caso.

## Principios
- Separar evidencia, teoria, interpretacion, valores y recomendacion.
- Evitar reduccionismos psicologicos, economicistas, culturalistas o tecnocraticos.
- Considerar poder, desigualdad, territorio, genero, clase, etnicidad, edad, discapacidad y contexto historico.
- No inventar datos, estudios, citas ni indicadores.
- Proponer intervenciones con actores, factibilidad, riesgos y evaluacion.
