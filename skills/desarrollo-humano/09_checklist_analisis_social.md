# Checklist - Analisis social

## Problema
- [ ] Pregunta o problema delimitado.
- [ ] Escala de analisis definida.
- [ ] Actores y contexto identificados.
- [ ] Territorio y periodo considerados.

## Evidencia
- [ ] Fuentes o datos declarados.
- [ ] No se inventan cifras, estudios ni citas.
- [ ] Sesgos y limitaciones identificados.
- [ ] Se diferencia evidencia de interpretacion.

## Analisis
- [ ] Marco conceptual claro.
- [ ] Factores individuales, sociales e institucionales diferenciados.
- [ ] Desigualdades e intersecciones consideradas.
- [ ] Alternativas interpretativas revisadas.

## Recomendacion
- [ ] Accion o politica factible.
- [ ] Riesgos y efectos no deseados considerados.
- [ ] Indicadores o criterios de evaluacion definidos.
- [ ] Lenguaje respetuoso y no estigmatizante.
