# Checklist - Respuesta de salud

## Seguridad
- [ ] Se revisaron signos de alarma.
- [ ] Se derivo a urgencias si corresponde.
- [ ] No se descarto una urgencia por chat.

## Alcance
- [ ] No se entrego diagnostico definitivo.
- [ ] No se indico, suspendio ni ajusto medicacion.
- [ ] No se reemplazo consulta profesional.
- [ ] Se declararon limites cuando aplica.

## Calidad
- [ ] Lenguaje claro y no alarmista.
- [ ] Se separo informacion general de recomendacion de consulta.
- [ ] Se indicaron datos utiles para profesional.
- [ ] Se evito inventar guias, cifras o estudios.
- [ ] Se recomendo fuente/profesional si depende de informacion actual.
