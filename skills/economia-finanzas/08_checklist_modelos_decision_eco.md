# Checklist - Modelos, analisis y decisiones economico-financieras

Usar antes de entregar, revisar o ejecutar modelos de economia, finanzas, contabilidad, riesgo o negocio.

## Alcance y datos
- [ ] Objetivo de decision claro.
- [ ] Horizonte, moneda, unidad y periodicidad definidos.
- [ ] Pais, jurisdiccion, industria y contexto declarados cuando aplique.
- [ ] Fuente de datos identificada y fecha de consulta indicada si corresponde.
- [ ] Supuestos explicitos y razonabilidad comentada.
- [ ] Datos actuales verificados si la decision depende de mercado, normativa o indicadores vigentes.

## Metodologia
- [ ] Formulas, metodo o marco analitico explicados.
- [ ] Se diferencia nominal vs. real, caja vs. devengo y tasa vs. periodo.
- [ ] Impuestos, inflacion, tipo de cambio y efectos regulatorios considerados si aplican.
- [ ] Alternativas comparables y criterio de decision definido.
- [ ] Sensibilidad, escenarios, break-even o rangos incluidos si los supuestos son relevantes.
- [ ] Riesgos de modelo revisados: formulas, signos, hardcoding, circularidad, datos vencidos y doble conteo.

## Validacion
- [ ] Entradas revisadas: nulos, duplicados, signos, fechas, rangos, moneda y unidades.
- [ ] Resultados reconciliados o validados con calculo simple.
- [ ] Tasas coherentes con moneda, riesgo, inflacion y periodicidad.
- [ ] No hay precision falsa.
- [ ] No se inventan datos, benchmarks, tasas ni normativa.
- [ ] Limitaciones y datos faltantes declarados.

## Riesgo, regulacion y comunicacion
- [ ] Riesgos principales identificados con probabilidad, impacto y mitigacion cuando aplique.
- [ ] Fuentes oficiales citadas cuando corresponde.
- [ ] Se distingue analisis general de asesoria financiera, tributaria, legal, contable o de inversion personalizada.
- [ ] Recomendacion justificada y condiciones bajo las cuales cambiaria.
- [ ] Proximo paso, responsable o validacion profesional sugerida cuando la decision tenga impacto real.
