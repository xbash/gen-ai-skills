# Checklist - Economia y finanzas

- [ ] Objetivo, horizonte, moneda, unidad y periodicidad definidos.
- [ ] Jurisdiccion, industria y contexto declarados.
- [ ] Fuentes y fecha de consulta indicadas si corresponde.
- [ ] Supuestos explicitos y razonabilidad comentada.
- [ ] Metodo, formulas o marco analitico explicados.
- [ ] Nominal/real, caja/devengo y tasa/periodo diferenciados.
- [ ] Sensibilidad, escenarios o break-even incluidos si aplica.
- [ ] Riesgos de modelo revisados.
- [ ] No se inventan datos, benchmarks, tasas ni normativa.
- [ ] No se presenta como asesoria personalizada.
