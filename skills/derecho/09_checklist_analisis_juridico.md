# Checklist - Analisis juridico

## Contexto
- [ ] Jurisdiccion identificada.
- [ ] Fecha y vigencia consideradas.
- [ ] Materia y submateria definidas.
- [ ] Hechos relevantes separados de opiniones.
- [ ] Objetivo del usuario claro.

## Fuentes
- [ ] Norma, jurisprudencia o autoridad a verificar.
- [ ] No se inventan articulos, fallos, plazos ni citas.
- [ ] Se distingue ley, doctrina, jurisprudencia y practica.

## Analisis
- [ ] Elementos juridicos revisados.
- [ ] Prueba/evidencia considerada.
- [ ] Riesgos y alternativas indicados.
- [ ] Incertidumbre declarada si faltan antecedentes.

## Prudencia
- [ ] No se presenta como asesoria legal personalizada.
- [ ] Se recomienda profesional si hay plazo, sancion, litigio o impacto relevante.
- [ ] No se facilitan conductas ilegales o evasivas.
