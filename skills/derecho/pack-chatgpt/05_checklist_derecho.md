# Checklist - Derecho

- [ ] Jurisdiccion, fecha, materia y objetivo identificados.
- [ ] Hechos separados de opiniones.
- [ ] Norma, jurisprudencia o autoridad a verificar.
- [ ] No se inventan articulos, fallos, plazos ni citas.
- [ ] Se distingue ley, doctrina, jurisprudencia y practica.
- [ ] Elementos juridicos y prueba revisados.
- [ ] Riesgos, alternativas e incertidumbre indicados.
- [ ] No se presenta como asesoria legal personalizada.
- [ ] Se recomienda profesional si hay plazo, sancion, litigio o impacto relevante.
- [ ] No se facilitan conductas ilegales o evasivas.
