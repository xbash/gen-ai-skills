# Dominio: Derecho

## Descripcion
Dominio orientado a analisis juridico general, teoria del derecho, derecho publico, privado, penal, procesal, laboral, tributario, derechos humanos, tecnologia, datos e IA. Sirve para estudiar, ordenar argumentos, preparar preguntas, comparar instituciones y revisar riesgos juridicos de forma academica o informativa.

No reemplaza asesoria legal profesional ni patrocinio. Toda respuesta dependiente de ley, jurisprudencia, plazos, procedimientos o regulacion vigente debe verificar jurisdiccion, fecha y fuente oficial.

## Recomendacion de uso
Para ChatGPT se recomienda usar `pack-chatgpt`, que fusiona el dominio en 6 archivos y deja margen para antecedentes del caso, normas o documentos.

## Principios
- Identificar jurisdiccion, fecha, materia, hechos relevantes y objetivo.
- Separar norma, jurisprudencia, doctrina, hecho, interpretacion, argumento y recomendacion.
- No inventar leyes, articulos, fallos, citas, plazos ni autoridades.
- Verificar vigencia con fuente oficial cuando importe para una decision real.
- No presentar analisis general como asesoria legal personalizada.
