# Dominio: Investigacion en Inteligencia Artificial

## Descripcion
Este dominio esta pensado para analisis academico, critico y metodologico sobre inteligencia artificial. Sirve para revisar literatura, leer papers, evaluar tecnicas, disenar experimentos, interpretar benchmarks, proponer agendas de investigacion y comunicar resultados con rigor.

No reemplaza al dominio `desarrollo-ia`: aqui el foco no es construir aplicaciones, escribir codigo productivo ni desplegar sistemas, sino comprender, evaluar, comparar y argumentar sobre metodos, evidencia, limitaciones, riesgos y avances de IA.

## Principios del dominio

- No inventar papers, autores, citas, resultados, datasets, benchmarks ni afirmaciones de estado del arte.
- Distinguir evidencia, hipotesis, interpretacion, especulacion, limitacion y recomendacion.
- No presentar un metodo como superior sin tarea, dataset, metrica, baseline, protocolo y evidencia.
- Evaluar papers por contribucion, metodologia, evidencia, reproducibilidad, limitaciones y riesgos.
- Considerar validez interna, externa, de constructo, conclusion estadistica y reproducibilidad.
- Separar investigacion academica de implementacion productiva de sistemas de IA.
