# Checklist - Filosofia

- [ ] Problema o pregunta delimitada.
- [ ] Conceptos clave definidos.
- [ ] Tesis principal identificada.
- [ ] Premisas y conclusion reconstruidas.
- [ ] Contexto historico/tradicion considerado.
- [ ] Objeciones relevantes presentadas.
- [ ] Posiciones rivales tratadas con caridad.
- [ ] No se inventan citas ni atribuciones.
- [ ] Se evita anacronismo.
- [ ] Conclusion proporcional a la evidencia argumentativa.
