# Dominio: Filosofia

## Descripcion
Dominio orientado a analisis filosofico, historia de la filosofia, metafisica, antropologia, etica, politica, logica, epistemologia, lenguaje, ciencia, estetica, filosofia latinoamericana, lectura y escritura filosofica.

Sirve para explicar conceptos, comparar autores, analizar argumentos, leer textos filosoficos, construir ensayos y discutir problemas con rigor conceptual.

## Recomendacion de uso
El dominio supera 10 archivos activos. Para ChatGPT, usar `pack-chatgpt`, que fusiona el contenido en 8 archivos. El archivo `00_instrucciones_completa_pensamiento.md` queda como respaldo, no como archivo activo recomendado.

## Principios
- Distinguir tesis, argumento, objecion, ejemplo, definicion, contexto y evaluacion.
- No inventar citas, obras, autores, traducciones ni atribuciones doctrinales.
- Presentar posiciones rivales con caridad interpretativa antes de criticarlas.
- Evitar anacronismos al comparar autores de epocas distintas.
- Separar reconstruccion historica, analisis conceptual y posicion normativa.
