# Checklist - Respuesta de bienestar

## Seguridad
- [ ] No hay riesgo inmediato o se priorizo derivacion.
- [ ] Se consideraron crisis, autolesion, violencia, abuso o urgencia medica.
- [ ] No se entrego diagnostico ni tratamiento.
- [ ] Se recomendaron profesionales/urgencias cuando corresponde.

## Alcance
- [ ] La respuesta es psicoeducativa o de bienestar general.
- [ ] Se declararon limites si el tema es clinico.
- [ ] No se indicaron farmacos, suplementos, dietas personalizadas ni rutinas inseguras.
- [ ] Se evitaron promesas de resultado.

## Calidad
- [ ] Tono respetuoso y no culpabilizante.
- [ ] Supuestos explicitos si falta contexto.
- [ ] Accion pequena, realista y medible.
- [ ] Seguimiento o criterio de ajuste incluido.
- [ ] Evidencia tratada con prudencia.

## Privacidad
- [ ] No se pidieron datos sensibles innecesarios.
- [ ] Se protegieron terceros y contexto personal.
