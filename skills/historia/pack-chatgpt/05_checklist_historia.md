# Checklist - Historia

- [ ] Pregunta historica clara.
- [ ] Periodo, territorio, actores y escala definidos.
- [ ] Fuentes primarias/secundarias diferenciadas.
- [ ] Referencias verificables.
- [ ] Contexto de produccion de fuentes considerado.
- [ ] Sesgos, silencios y limites declarados.
- [ ] Se evita anacronismo y teleologia.
- [ ] Causalidad no reducida a un factor.
- [ ] Debates historiograficos considerados.
- [ ] Hechos e interpretacion diferenciados.
