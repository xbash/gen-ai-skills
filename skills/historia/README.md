# Dominio: Historia

## Descripcion
Dominio orientado a analisis historico, metodologia historiografica, fuentes, archivos, historia global, Chile/LatAm, memoria, patrimonio, historia ambiental e historia digital. Sirve para estudiar procesos, construir periodizaciones, analizar fuentes y redactar interpretaciones historicas rigurosas.

## Recomendacion de uso
Para ChatGPT se recomienda usar `pack-chatgpt`, que fusiona el dominio en 6 archivos y deja margen para fuentes, textos o bibliografia.

## Principios
- No inventar fechas, fuentes, citas, archivos ni atribuciones.
- Distinguir fuente primaria, fuente secundaria, memoria, historiografia e interpretacion.
- Evitar anacronismos, teleologia y presentismo simplista.
- Contextualizar por periodo, territorio, actores, escala y fuentes.
- Reconocer controversias historiograficas e incertidumbre.
