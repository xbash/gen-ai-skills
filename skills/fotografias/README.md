# Dominio: Fotografias

## Descripcion
Dominio orientado a mejora fotografica conservadora, restauracion digital, retratos, producto, color, luz, textura, fondo, composicion e integridad visual. Su principio central es que la imagen mejorada debe seguir siendo la misma fotografia, solo tecnicamente optimizada.

No esta pensado para redisenar escenas, alterar identidades, crear personas nuevas, falsificar evidencia visual ni convertir fotografia documental en fantasia salvo que el usuario pida expresamente una edicion transformativa y sea apropiada.

## Recomendacion de uso
Para proyectos con limite de archivos, cargar `00` a `06`. El archivo `proyecto_mejora_fotografica_archivos_md.md` es respaldo extendido y puede omitirse.

## Principios
- Preservar identidad, anatomia, composicion, perspectiva, luz y atmosfera original.
- Mejorar nitidez, ruido, compresion, color y detalle sin inventar contenido relevante.
- No alterar evidencia, documentos, rostros, marcas, productos o escenas de forma enganosa.
- Declarar si una edicion es restauracion, mejora tecnica, retoque estetico o transformacion creativa.
- Respetar derechos, consentimiento, privacidad y sensibilidad de imagenes personales o historicas.
