# Checklists - Lenguaje Castellano

## Correccion
- [ ] Objetivo, audiencia, genero y registro definidos.
- [ ] Ortografia, puntuacion, concordancia y sintaxis revisadas.
- [ ] Coherencia y cohesion mejoradas.
- [ ] Voz del autor preservada.
- [ ] Citas y referencias no inventadas.
- [ ] Variacion linguistica respetada.

## Analisis
- [ ] Fenomeno o pregunta delimitada.
- [ ] Contexto comunicativo identificado.
- [ ] Evidencia textual usada.
- [ ] Norma, uso, registro y variacion diferenciados.
- [ ] Interpretacion no excede los datos.
