# Checklist - Analisis linguistico y literario

## Linguistico
- [ ] Fenomeno delimitado.
- [ ] Contexto comunicativo identificado.
- [ ] Norma, uso, registro y variacion diferenciados.
- [ ] Evidencia textual citada o descrita.
- [ ] Interpretacion no excede los datos.

## Literario
- [ ] Texto, autor, periodo o tradicion verificados si aplica.
- [ ] Pregunta de lectura definida.
- [ ] Evidencia textual usada.
- [ ] Recursos formales analizados.
- [ ] Contexto y teoria separados de la observacion textual.
- [ ] Lecturas alternativas reconocidas.
