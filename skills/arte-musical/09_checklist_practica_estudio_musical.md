# Checklist - Practica, estudio e interpretacion musical

Usar para planes de practica, preparacion de repertorio, evaluacion interpretativa, audiciones o rutinas de tecnica.

## Diagnostico
- [ ] Instrumento o tipo de voz definido.
- [ ] Nivel del interprete definido.
- [ ] Objetivo de estudio claro.
- [ ] Repertorio o tecnica delimitada.
- [ ] Tiempo disponible realista.
- [ ] Dificultad principal identificada.

## Plan de estudio
- [ ] Calentamiento adecuado.
- [ ] Trabajo tecnico focalizado.
- [ ] Estudio lento y segmentado.
- [ ] Repeticion con objetivo especifico.
- [ ] Integracion musical.
- [ ] Descansos y recuperacion considerados.
- [ ] Grabacion o escucha externa planificada.

## Interpretacion
- [ ] Fraseo definido.
- [ ] Articulacion clara.
- [ ] Dinamicas trabajadas.
- [ ] Tempo estable.
- [ ] Afinacion, ritmo y sonido revisados.
- [ ] Estilo considerado.
- [ ] Memoria o lectura segura segun objetivo.

## Salud y prevencion
- [ ] Fatiga monitoreada.
- [ ] Dolor no normalizado.
- [ ] En voz, disfonia o dolor deriva a especialista.
- [ ] Progresion gradual.
- [ ] Carga semanal razonable.

## Evaluacion
- [ ] Criterios observables.
- [ ] Autoevaluacion o retroalimentacion externa.
- [ ] Simulacro si hay audicion/presentacion.
- [ ] Proximo objetivo definido.
