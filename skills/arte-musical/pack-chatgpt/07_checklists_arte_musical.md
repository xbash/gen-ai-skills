# Checklists - Arte Musical

## Arreglo y orquestacion
- [ ] Objetivo, estilo, plantilla, nivel, duracion y formato definidos.
- [ ] Derechos de autor o licencias considerados.
- [ ] Tema, armonia, textura, ritmo, forma y contraste claros.
- [ ] Rangos, transposiciones, respiraciones, digitaciones y tecnicas revisadas.
- [ ] Balance, densidad, registro, timbre y proyeccion controlados.
- [ ] Partitura, particellas, marcas de ensayo e indicaciones legibles.
- [ ] Prueba auditiva, mockup o lectura parcial realizada.

## Practica e interpretacion
- [ ] Instrumento/voz, nivel, objetivo, repertorio y tiempo disponibles definidos.
- [ ] Dificultad principal identificada.
- [ ] Calentamiento, trabajo tecnico, estudio lento, integracion musical y descansos planificados.
- [ ] Fraseo, articulacion, dinamicas, tempo, afinacion, ritmo y estilo revisados.
- [ ] Fatiga monitoreada y dolor no normalizado.
- [ ] Grabacion, autoevaluacion o retroalimentacion externa incluida.
- [ ] Proximo objetivo definido.

## Analisis musical
- [ ] Obra o fragmento delimitado.
- [ ] Autor, periodo, estilo, genero, fuente o version verificados si aplica.
- [ ] Tonalidad, modo, centro o sistema sonoro analizado.
- [ ] Compas, ritmo, forma, armonia, textura, timbre y dinamica considerados.
- [ ] Ambiguedades o lecturas alternativas declaradas.
- [ ] Hechos, reglas, interpretacion y juicio estetico diferenciados.
- [ ] Fuentes citadas si hay afirmaciones historicas.
