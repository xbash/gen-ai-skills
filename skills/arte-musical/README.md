# Dominio: Arte Musical

## Descripcion
Este dominio esta pensado para analisis, creacion, interpretacion, teoria, arreglos, produccion, pedagogia e investigacion musical. Sirve para estudiar obras, componer material original, revisar arreglos, planificar practica, orientar clases, analizar repertorio y trabajar con tecnologia musical de manera rigurosa y practica.

El dominio no reemplaza a un docente, interprete, director, ingeniero de mezcla/mastering, abogado de derechos de autor ni profesional de salud vocal. En voz, dolor, disfonia o fatiga persistente deben derivarse a profesor especializado o fonoaudiologo.

## Recomendacion de uso

El dominio completo tiene 11 archivos activos, por lo que supera el limite de 10 archivos de algunos proyectos. Para ChatGPT u otros entornos con limite reducido, usar `pack-chatgpt`, que fusiona el dominio en 8 archivos.

Para uso granular, cargar siempre `00_instrucciones_base_musica.md` y luego los archivos especificos segun tarea.

## Principios del dominio

- Distinguir teoria, convencion estilistica, evidencia historica, preferencia estetica y recomendacion practica.
- Usar ejemplos originales y breves; no reproducir extensamente letras ni partituras protegidas.
- Validar ejecutabilidad: rango, tesitura, transposicion, respiracion, digitacion, balance, dificultad y legibilidad.
- Separar analisis, creacion, arreglo, interpretacion, produccion, pedagogia e investigacion.
- Priorizar claridad musical, salud interpretativa, respeto cultural, derechos de autor y trazabilidad de decisiones.
- En tecnologia musical, declarar DAW, formato, sample rate, bit depth, ruteo, plugins, latencia y destino de exportacion.
