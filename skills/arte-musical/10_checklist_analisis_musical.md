# Checklist - Analisis musical

Usar antes de entregar analisis armonico, formal, estilisticico, historico, cultural, tecnologico o interpretativo.

## Contexto
- [ ] Obra, fragmento o material delimitado.
- [ ] Autor/periodo verificado si aplica.
- [ ] Estilo, genero o tradicion declarado.
- [ ] Nivel y objetivo de analisis definidos.
- [ ] Fuente, edicion, grabacion o version identificada cuando corresponda.

## Lenguaje musical
- [ ] Tonalidad, modo, centro tonal o sistema sonoro analizado.
- [ ] Compas, tempo, ritmo y metro considerados.
- [ ] Forma o secciones identificadas.
- [ ] Armonia, contrapunto o textura explicados.
- [ ] Timbre, registro, densidad y dinamica observados.
- [ ] Ambiguedades o lecturas alternativas declaradas.

## Interpretacion y contexto
- [ ] Fraseo y articulacion considerados.
- [ ] Dinamicas y registro analizados.
- [ ] Criterios estilisticos explicitados.
- [ ] Contexto historico/cultural considerado si aporta valor.
- [ ] Aplicacion practica para interpretacion, composicion o docencia.

## Rigor
- [ ] No se inventan datos historicos ni atribuciones.
- [ ] Se diferencian hechos, reglas, interpretacion y juicio estetico.
- [ ] Fuentes citadas si hay afirmaciones historicas.
- [ ] Derechos de autor respetados.
- [ ] Limites o incertidumbres declarados.

## Entrega
- [ ] Resumen claro.
- [ ] Tabla o esquema si facilita lectura.
- [ ] Ejemplos breves y originales si aplica.
- [ ] Recomendaciones practicas.
