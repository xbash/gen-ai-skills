# Checklist - Arreglo, instrumentacion y orquestacion

Usar antes de entregar o revisar arreglos, adaptaciones, reducciones, orquestaciones o musica para ensamble/medios.

## Alcance
- [ ] Objetivo musical claro.
- [ ] Estilo, contexto y funcion definidos.
- [ ] Plantilla instrumental/vocal definida.
- [ ] Nivel de interpretes considerado.
- [ ] Duracion, forma y formato de entrega estimados.
- [ ] Derechos de autor, licencias o restricciones de material consideradas.

## Material musical
- [ ] Tema, motivo o gesto principal claro.
- [ ] Armonia o sistema sonoro coherente.
- [ ] Textura definida.
- [ ] Ritmo legible y funcional.
- [ ] Forma comprensible.
- [ ] Contraste y continuidad equilibrados.

## Ejecutabilidad
- [ ] Rangos comodos verificados.
- [ ] Transposiciones correctas.
- [ ] Respiraciones consideradas.
- [ ] Digitaciones, posiciones o tecnicas criticas revisadas.
- [ ] Dificultad adecuada al nivel.
- [ ] Articulaciones, dinamicas y cambios de tecnica claros.

## Balance y sonoridad
- [ ] Melodia o capa principal audible.
- [ ] Acompanamiento no satura.
- [ ] Registro equilibrado.
- [ ] Densidad controlada.
- [ ] Mezcla timbrica intencional.
- [ ] Prueba auditiva, mockup o lectura parcial realizada.

## Entrega
- [ ] Partitura legible.
- [ ] Particellas consistentes.
- [ ] Tempo, compas, tonalidad/centro y marcas de ensayo claras.
- [ ] Indicaciones de interpretacion suficientes.
- [ ] Validacion de ensayo o siguiente revision sugerida.
